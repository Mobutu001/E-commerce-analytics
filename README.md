# E-Commerce Sales Analytics (SQL + Power BI)

## Project Overview
This project analyzes an e-commerce marketplace dataset to understand sales performance, customer behaviour, and delivery efficiency.

## Tools Used
- SQL Server
- Power BI
- DAX

## Key Business Questions
- Which product categories generate the most revenue?
- Which customers contribute the highest spending?
- How does delivery time affect customer reviews?
- Which regions experience the slowest deliveries?

## Key Insights
- Revenue is concentrated in a small number of product categories.
- São Paulo generates the largest share of customers and revenue.
- Faster deliveries strongly correlate with higher review scores.
- Credit cards dominate payment methods.

## Dashboard
The Power BI dashboard provides interactive insights across four main areas:
- Executive Overview
- Customer Insights
- Logistics & Delivery Analysis
- Sales Trends
